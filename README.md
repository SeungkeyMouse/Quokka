## 구현 기능

### 기능 1
- 닮은 꼴 데이터 셋을 통해 해당 데이터와 비슷하다면 특정 쿼카 이미지를 보여줌.
### 기능 2
- 결과지를 카카오 메시지 템플릿을 통해 공유 가능.
### 기능 3
- 광고 스크립트를 삽입하여 특정 액션마다 광고 배너가 출력.
